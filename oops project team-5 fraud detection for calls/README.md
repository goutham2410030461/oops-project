# Carbon Emission Tracker

## Overview

**Carbon Emission Tracker** (also known as **CarbonSense**) is a desktop application designed to help individuals track, analyze, and reduce their carbon footprint. The application focuses on three major categories of emissions: Transport, Energy, and Food. By providing real-time tracking and personalized recommendations, it empowers users to make informed decisions about their environmental impact.

## Features

- **Real-time Tracking**: Instant emission calculations as users input data.
- **Visual Analytics**: Interactive pie charts and statistics to visualize emission distribution.
- **Personalized Tips**: Smart recommendations based on user behavior and data.
- **Modern UI**: A sleek dark theme with vibrant accent colors.
- **Educational Content**: Information on emission factors and reduction strategies.

## Installation Instructions

### Prerequisites

1. **Java Development Kit (JDK)**: Version 17 or higher.
2. **Apache Maven**: For managing dependencies and building the project.

### Quick Setup

1. Clone the repository:
   ```
   git clone https://github.com/YOUR_USERNAME/carbon-emission-tracker.git
   cd carbon-emission-tracker
   ```

2. Build the project using Maven:
   ```
   mvn clean install
   ```

3. Run the application:
   ```
   mvn javafx:run
   ```

## Usage Guidelines

1. **Dashboard**: View overall carbon footprint and identify the highest emission sources.
2. **Transport Tab**: Input distances traveled by car, bus, train, or flight to track transportation emissions.
3. **Energy Tab**: Enter household energy consumption to monitor energy-related emissions.
4. **Food Tab**: Track dietary choices and their impact on carbon emissions.
